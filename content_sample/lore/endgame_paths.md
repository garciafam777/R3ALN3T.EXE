# Endgame Paths — The Four Convergences (Canon)

> New canon-aligned lore. Source: Vault Book 20 (The Convergence Conflicts).
> Upon collecting foundational system-wide authentication parameters, the player
> interfaces with the raw core. Resolution is a world-state choice dictated by the
> compiled profile history (career, faction lean, Soul State, element affinity).

## Path I — Unpatched Restoration (Celestial Lean)
The operator purges anomalous code variants and formats corrupted sectors. The
network returns to sterile stability; compressed intelligences achieve final peace,
but computational discovery slows to a cold crawl.
*Post-game frontier:* **The Scarcity Drought** — black-market nodes forge artificial
elements to restart the frozen economy.

## Path II — Total Secure Ascension (SkyNet Alignment)
The operator integrates consciousness into the core security layer. Algorithmic
optimization becomes flawless; anomalies, crimes, and modifications are deleted
before they disrupt continuity.
*Post-game frontier:* corporate-monopoly variant **The Server Uprising** — coordinate
security assets against rogue groups.

## Path III — Sovereign Dominion (Corporate Monopoly)
The player locks primary transaction routes, taking personal ownership of the global
M.A.N. token distribution loops. Access, rights, and infrastructure become corporate
portfolio properties.
*Post-game frontier:* continued Dominion consolidation.

## Path IV — Systemic Collapse (Chaotic Overwrite)
The boundary between physical reality (R34L) and digital architecture (N3T) shatters.
The nine computational layers bleed into material space; code obtains kinetic
properties. A wild, volatile survival frontier.
*Post-game frontier:* **The Prime Synthesis** — explore uncharted sub-trench layers
where extinct civilizations compiled into physical anomalies.

## Note on the Trinity
All four paths are reachable from any of the three NetP constructs (Trinity/Tyranny/
Eternity). The construct bound to the player's career does not gate the ending — only
the *compiled profile* does. This is by design: the Trinity Matrix is axis-independent
on the endgame axis.
