# Player Avatar Sprites

R34L (physical) and N3T (jacked-in) versions of the main character.

## Naming convention

`player_{realm}_{pose}.svg`

| File | Usage |
|------|-------|
| `player_r34l_idle.svg` | Physical R34L Room |
| `player_r34l_walk.svg` | Physical movement |
| `player_n3t_idle.svg` | N3T mirror avatar |
| `player_n3t_combat.svg` | N3T battle pose |

## Resolver

`NetPSpriteResolver.player_avatar_path("r34l", "idle")`
