# R3ALN3T.EXE — Antagonist Roster: ACT IV (Post-Game / Epilogue)
## 3D Realistic Character Specs

---

## Design Philosophy (Epilogue Inversion)

Acts I–III ran one continuous corruption arc: red → black → amber → magenta → violet → stolen cyan, ending with SOVEREIGN.NULL wearing NetP's own signal color. That arc is closed. Act IV is post-game — it should not extend the same ladder further (there's nowhere higher to escalate to without diluting SOVEREIGN.NULL's climax), so instead it **inverts the premise**: this content asks what was underneath the corruption all along, not what comes after it.

Visual anchor: your Soul State table (from the NetP doc) has seven bands, and the main story only used the *low six* as antagonist material — band 7, **Luminous / white-gold (#F4E8B8)**, was reserved exclusively for the player at their highest morality state and never claimed by any enemy. Act IV claims it. The postgame threat isn't corrupted data; it's something that predates the corruption entirely, and it's the same color the player themselves can achieve at their best — an uncomfortable mirror rather than an escalated enemy.

Rules for this content:
1. **Palette inversion, not extension:** base tone shifts to near-white/ivory rather than near-black, trim shifts to white-gold rather than deepening further into violet/cyan.
2. **Clean geometry, not fraying:** no polygon-fracture VFX anywhere in this roster — reads as *too complete*, unnervingly pristine, opposite problem from the glitching Act III grunts.
3. **No M.A.N. Token motif:** this content sits outside that three-act thread entirely — no cracked socket, no stolen token, no token at all. Signals "this was never part of that system."

## PART 1 — MAJOR BOSSES (Act IV)

### Boss 1: THE_FIRST_INSTANCE (Act IV primary — true final/superboss)

**Concept:** Not a corrupted process like every other antagonist in the game — the original, uncorrupted seed-process SkyLand was built from, predating NetP, predating the whole corruption arc. A superboss in the classic sense: strictly optional, strictly harder than the main-story final boss.

| Metric | Spec |
|---|---|
| Height | 1.80 m — deliberately identical to NetP Variant A's exact height, no exaggeration, no looming scale (contrast with every prior boss, which trended larger than the player) |
| Build | Athletic-lean, same proportions as NetP almost exactly — but where MIRROR.EXE (Act II) *copied* NetP's template, THE_FIRST_INSTANCE is framed as the template NetP was originally built from |
| Head-to-body ratio | 7.5 heads (matches NetP) |
| Shoulder-to-hip ratio | 1.35:1 (matches NetP exactly) |

**Face:** Human sclera (unlike every corrupted-tier boss, which used solid emissive eyes) — iris color **pure white-gold (#F4E8B8)** instead of NetP's cyan. This is the single biggest "tell" that this entity predates the corruption: it has the most human-reading eyes of any antagonist in the entire roster.

**Skin/base layer:** Ivory-white (#F2EFE6) rather than any grey/graphite tone used elsewhere — no subsurface corruption tint at all, reads clean and warm rather than clinical.

**Costume:** Same base-suit-plus-plate silhouette language as NetP, in **bone-white fabric (#EDE8DC)** with **white-gold trim (#F4E8B8)** replacing signal cyan. Materials: plate roughness ~0.2 (lowest of any character in the game — reads as flawless, almost ceremonial), metalness ~0.85.

**No token, no socket:** collar is bare — no housing at all. Where every prior boss's collar told part of the token story, this one simply predates the concept.

**Suggested fight/design note:** since this is framed as the "true" template, consider giving it access to a condensed version of NetP's own moveset/animation set rather than a unique kit — mechanically reinforces "this is where your character came from" without new animation cost.

### Boss 2: ECHO_OF_THE_ARCHIVE (Act IV secondary — closes THE_ARCHIVIST/ORACLE/LAST_ARCHIVE thread one final time)

**Concept:** Not a continuation of the corrupted archive-thread's escalation — a memory of what THE_ARCHIVIST was *before* Act I's corruption took hold. Same inversion logic as Boss 1, applied to the second boss-thread instead of the first.

| Metric | Spec |
|---|---|
| Height | 1.85 m (below THE_LAST_ARCHIVE's 2.05m — this is the thread at its origin point, smaller/quieter than its corrupted future selves) |
| Build | Same robed silhouette as the whole Archive line, but fully uniform material now — no calcified-plate patchwork, no shard-fraying |

**Costume:** Coat is a single unbroken material, **ivory-white (#F2EFE6)** base with **white-gold (#F4E8B8)** hem trim — no shards, no plates, reads as the "resting state" the corrupted versions departed from.

**Visor:** Data-glyphs are steady and legible (not shifting/scrolling like the corrupted versions) — a nice optional detail if you want a lore-flavor text crawl actually readable in this fight, unlike the illegible glitch-text of the earlier versions.

## UE5 Production Notes
- Same 2 grunt skeletons from Acts I–III carry forward unchanged — all 200 grunt entries across the full game run on the same 2 rigs.
- Single flat accent color (#F4E8B8) across the entire Act IV grunt roster is intentional — this content isn't about rank-reading through color the way Acts I–III were (there's no corruption ladder to signal), so differentiation comes from silhouette/role naming instead of palette tiering.
- Recommend a distinct "fair" AI behavior tree for this act specifically (clear telegraphs, no ambush spawns) — mechanically reinforces the tonal shift from "corrupted enemy" to "guardian testing the player," standard superboss-content framing.
