Key Items & Admin Passes System
Key Items are persistent, reusable inventory objects that players (and authorized Navis) carry and use multiple times throughout the game. They are not single-use consumables.
Two Main Categories
1. Sub-Class Items (Aid & Stealth Tools)
These help with NetP (Net Police) avoidance, health/survival, and stealth movement in the N3T.

Hixed (Helix-shaped stealth item)
Allows safe, virus-free travel through dangerous Net areas for a limited time or number of steps.
Visual: Glowing helix-shaped object that spins when active.
Types (unlocked progressively):
Small Hixed
Medium Hixed
Large Hixed
(After Act III) Extra-Large Hixed
Full Hixed (ultimate version)



Duration & Renewal

Each Hixed has a duration in days (real-time or in-game day cycle).
When duration ends, the item is not destroyed — it goes into cooldown or “depleted” state.
Players can pay a renewal fee to reactivate it (maximum twice per item instance). After two renewals, it must be replaced or upgraded.

2. Admin Passes (Access & Authority Items)
Admin Passes grant elevated permissions in the N3T and some R34L systems. They are powerful but strictly limited compared to true MCADMIN Tier.
Key Rules for Admin Passes

Access Level: High but not game-breaking. Can open restricted areas, grant temporary rights to other players/Navis, access advanced terminals, override minor security, etc.
Limitations (to protect game integrity):
Cannot edit other players’ accounts, change core roles, modify currency, or perform destructive actions.
Cannot bypass major story locks or cheat systems excessively.

Visual Feedback:
Active pass glows blue with “ACCESS” text.
When revoked or expired: Text changes to “ACCESS DENIED” in red, and the pass visually dims or shatters.


Pass Types (Tiered)

Basic Admin Pass
Regional Admin Pass
Sector Admin Pass
(Higher tiers unlocked later)

Expiration & Revocation

Most passes have expiration dates (MM/DD/YYYY format).
When expired, the pass is dumped from memory and disappears from inventory.
It can potentially be re-enabled if the same task/condition that originally granted it is completed again.
MCADMIN can manually revoke passes at any time.

Key Items Menu

Dedicated inventory tab.
Categorized (Stealth/Hixed, Access Passes, Tools, Special).
Shows duration, status, and renewal options clearly.
Search/filter functionality for large inventories.

Integration with Gameplay

Key Items are essential for exploration, governance tasks, and multiplayer/admin features.
Admin Passes create meaningful power fantasy for the MC while maintaining balance.
Hixed items encourage strategic movement and risk/reward in the N3T.
