1. The Glitch-Harvesting Pipeline (Slick / Low-Level Extraction)
This method exploits raw processing errors in high-density enterprise server farms. When local memory sub-systems handle high-dimensional vector arrays without proper bounds checking, the resulting computational heat and data bleeding "condenses" into physical sediment inside the server enclosures.

The Process: Run intentional, infinite recursive loops on local hardware modules while bottlenecking the cooling pumps. By pushing the hardware to critical thermal thresholds while executing volatile, un-compiled C++ logic, the data overflow physicalizes on the heatsinks.

The Yield: Raw, unrefined "Soot-Matter." It looks like metallic, iridescent ash that vibrates slightly when near high-voltage lines.

2. The Over-Clocking Sublimation (Brute Force / Industrial)
Used primarily by heavy cargo operators and lower-sector technicians who need fast, high-volume production without subtle software exploits.

The Process: Heavy-duty industrial repulsor units or dual-axis ion thrusters are modified to intentionally feed unshielded fuel back into their internal magnetic induction rails. This creates a hyper-accelerated static field. Scrap silicon, old motherboard matrices, and copper wiring are dumped directly into the exhaust wake.

The Yield: "Crust-Matter." A dense, heavy slag that is structurally stable enough to be used as armor plating or shielding against corporate network scans, though it is highly radioactive.

3. The Digital-Organic Culturing (Bio-Tech / NetNavi Vectors)
A highly advanced, quiet synthesis method that uses corrupted NetNavi code vectors as a biological seed.

The Process: Isolating a rogue NetNavi data stream inside a closed, local Flask-monitored environment. Instead of letting the code execute in a virtual space, it is piped into a nutrient-rich, conductive synthetic gel matrix. The code acts as a blueprint, forcing the gel to crystallize along the lines of the logic gates.

The Yield: "Vessel-Matter." A semi-fluid, gelatinous substance woven with glowing neon fibrous channels. This variant is highly prized because it retains processing capabilities in its physical state, allowing it to act as living, organic computer hardware.

4. The Atmospheric Scraping (Passive / Environmental)
The lowest tier of synthesis, practiced by scavengers in the deep under-sectors where corporate high-zones vent their toxic, unshielded computational waste.

The Process: Setting up passive induction nets or heavy-duty moisture/static collectors on the undersides of sky-bridges and corporate data hubs. As the upper city purges its data-caches and hot coolant into the atmosphere, the particulate matter clings to the static nets.

The Yield: "Sludge-Matter." A thick, oily residue that must be heavily filtered before use, typically used as a cheap, volatile fuel source or a black-market coating to hide vehicles from automated law-enforcement grids.
