# UI Sprite Icons

Icons for PED interfaces, encrypt/decrypt flow, and currency display.

## Naming convention

`icon_{name}.svg`

| File | Usage |
|------|-------|
| `icon_encrypt.svg` | Jack in / encrypt sequence |
| `icon_decrypt.svg` | Return to R34L / decrypt |
| `icon_man_helix.svg` | M.A.N. balance currency |

## Resolver

`NetPSpriteResolver.ui_icon_path("icon_encrypt")`
