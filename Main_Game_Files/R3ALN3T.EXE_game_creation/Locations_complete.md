# R3ALN3T Game Locations Registry

This document serves as the master registry for all locations within the R3ALN3T universe, spanning both the Physical (r34l) and Network (n3t) layers.

## Overview
- **Physical Layer (r34l):** Areas experienced in first-person/third-person gameplay.
- **Network Layer (n3t):** Virtual grids and cyberspace sectors accessed via terminals.

## Master Location List

### 1. Physical Layer (r34l)
- **[[r34l_room]]**: The player's primary living space and terminal access point.
- **[[NetP_Headquarters]]**: The central hub for NetP administration.
- **[[Cyber_Square]]**: Public urban hub for faction interactions.
- **[[Sector_01_Slums]]**: High-corruption residential district.
- **[[Sector_04_Industrial]]**: Heavily guarded, high-heat zone.
- **[[LunaMars_District]]**: High-end corporate/residential area.
- **[[Heaven_District]]**: Elite governance sector.
- **[[Hell_District]]**: Ruined sector, access restricted.

### 2. Network Layer (n3t)
- **[[Net_Room]]**: The digital manifestation of the r34l_room.
- **[[Grid_Sector_01]]**: Entry-level combat grid.
- **[[Grid_Sector_02]]**: Moderate challenge grid.
- **[[Grid_Sector_03]]**: High-heat combat grid.
- **[[Grid_Sector_04]]**: Boss encounter grid.
- **[[Grid_Sector_05]]**: The "Deep Web" core interface.
- **[[MCADMIN_Core]]**: The seat of the Masked Admin's power.
